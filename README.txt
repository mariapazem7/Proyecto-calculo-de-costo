

Proyecto calculo de costo v1.0 creado por Maria Paz Faundez


El siguiente proyecto tiene como objetivo crear un algoritmo en pseudocodigo que permita simular el calculo del costo de un producto.
Para lograrlo se utilizaron las herramientas aprendidas durante el primer modulo del Boot Camp de DWFS C17 de la UDD.

Algunos contenidos que se pueden encontrar son:

1. Operaciones aritméticas y operadores de comparación	
2. Implementación de control de flujo: condiciones y bucles	
3. Uso de arreglos unidimensionales
